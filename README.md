# Healthcare Revenue Analysis (SQL + Pandas + Power BI)

## Overview

End-to-end revenue analysis project on hospital dataset using SQL for querying, Pandas for data cleaning, and Power BI for visualization.

---

## Objective

* Analyze total revenue distribution
* Identify top 10% revenue contributors
* Examine cost patterns across departments

---

## Tools Used

* SQL
* Python (Pandas)
* Power BI

---

## Key Metrics

* Total Revenue
* Top 10% Revenue
* Remaining 90% Revenue
* Average Treatment Cost
* Patient Count
